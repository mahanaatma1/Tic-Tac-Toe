Tic-Tac-Toe Game
Welcome to my Tic-Tac-Toe game! This classic game is built using pure HTML, CSS, and JavaScript, providing a simple yet engaging user experience.

Features
Intuitive User Interface: The game board is designed with simplicity and ease of use in mind, ensuring that players of all ages can enjoy the game.
Two-Player Mode: The game supports a traditional two-player mode where each player takes turns to mark their symbol (X or O) on the 3x3 grid.
Dynamic Game Logic: JavaScript handles all game logic, including player turns, win conditions, and detecting Draw.
Restart Button: Players can easily restart the game at any time with the click of a button.

How to Play
Start the Game: The game begins with an empty 3x3 grid.
Player Turns: Players take turns clicking on an empty cell to place their symbol (X or O). Player X always goes first.
Win Conditions: The game checks for three consecutive symbols horizontally, vertically, or diagonally. If a player achieves this, they win the game.
Tie Condition: If all cells are filled without any player achieving three consecutive symbols, the game ends in a tie.
Restart: Click the restart button to clear the board and start a new game.
Technical Details
HTML: Structures the game board and provides the basic layout.
CSS: Styles the game board, cells, and buttons, and handles the winning animation.
JavaScript: Manages the game logic, including player turns, win detection, and resetting the game.
